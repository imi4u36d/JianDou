# 产品范围说明

## 1. 文档目的

本文档定义 `AI Cut` 首版的产品范围，用于约束当前阶段的实现、联调和验收口径。首版目标不是完整的智能投放系统，而是先稳定产出可下载、可复用的短剧素材。

## 2. MVP 范围

### 2.1 必须实现

- `本地视频上传`
  - 支持上传原始短剧视频作为输入素材。
  - 支持基础文件校验与存储。

- `剪辑任务创建`
  - 用户可以创建一个剪辑任务，并为任务填写参数。
  - 任务至少包含：目标时长范围、产出数量限制、平台/比例要求、片头模板、片尾模板。

- `模型辅助剪辑规划`
  - 首版支持 `qwen` 作为模型 provider。
  - 模型用于辅助识别可剪辑片段、生成候选方案或剪辑建议。
  - 当模型结果不可用时，允许使用规则或启发式策略兜底。

- `自动裁切与拼接`
  - 基于任务参数生成候选片段。
  - 支持按目标时长范围输出素材。
  - 支持片头、片尾效果拼接。

- `素材产出与下载`
  - 支持生成多个候选素材。
  - 支持预览生成结果。
  - 支持下载最终素材文件。

- `任务状态管理`
  - 至少支持：待处理、处理中、成功、失败、部分成功。
  - 支持查看失败原因或处理摘要。

### 2.2 首版允许的能力

- 单视频单任务优先，后续再扩展批量处理。
- 模板化片头片尾，先提供固定效果或少量可配置效果。
- 以“稳定可产出”为目标，不强求每次都生成最优创意素材。

## 3. 用户流程

### 3.1 标准流程

1. 用户上传原始视频。
2. 用户填写剪辑参数。
3. 系统校验输入并创建剪辑任务。
4. 系统调用模型和视频处理链路生成候选片段。
5. 系统按时长范围与产出数量限制生成素材。
6. 系统为素材拼接片头、片尾效果。
7. 用户预览结果并选择保留素材。
8. 用户下载最终素材用于后续投放或分发。

### 3.2 异常流程

- 上传失败时，提示文件格式、大小或网络错误。
- 剪辑失败时，保留失败原因，允许重新发起任务。
- 生成数量不足时，明确标记为部分成功，而不是静默返回空结果。

## 4. 已确认需求

- 需要支持根据用户提供的视频和参数进行短剧剪辑。
- 需要按参数控制素材时长范围。
- 需要限制单次产出数量。
- 需要支持片头和片尾效果拼接。
- 需要生成可用于后续投放的素材文件。
- 首版模型 provider 只要求支持 `qwen`。
- 当前阶段不实现投放到各平台的完整发布链路。

## 5. 暂不做内容

- 平台投放发布
  - 不做各内容平台的自动发布、账号绑定、授权管理和投放调度。

- 复杂创意策略
  - 不做完整的爆款预测、自动选题、自动投流优化。

- 高级剪辑能力
  - 不做逐帧精修、复杂动效、镜头级手工编辑、专业调色、复杂音频混音。

- 多源内容编排
  - 不做多视频混剪、批量多任务编排、跨项目聚合管理。

- 数据闭环优化
  - 不做投放效果回流、转化归因、自动 A/B 测试闭环。

- 多模型多 Provider 首版支持
  - 首版只保证 `qwen` provider 可用，其他模型提供方后续再扩展。

## 6. 配置要求

### 6.1 运行环境配置

- 服务端需要可配置的 API 地址、端口和前端跨域地址。
- 需要配置本地数据库和 Redis，用于任务、状态和运行时数据管理。
- 需要配置本地存储目录，用于上传、处理中间文件和输出文件。

### 6.2 模型配置

- `model.provider` 首版固定支持 `qwen`。
- 需要配置 `model.model_name`、`model.endpoint`、`model.api_key` 和 `model.timeout_seconds`。
- 模型调用超时需要可控，避免单个任务长时间阻塞。

### 6.3 剪辑管线配置

- 需要支持默认画幅比例配置，首版以 `9:16` 为主。
- 需要支持单任务最大产出数量限制。
- 需要支持单个源视频的最大处理时长限制。
- 需要支持默认片头模板和片尾模板配置。

### 6.4 存储要求

- 上传文件、输出文件和临时文件需要分目录管理。
- 输出文件需要可通过统一的公开地址访问或下载。
- 临时文件需要在任务完成后可清理，避免磁盘无限增长。

## 7. 第一阶段验收口径

- 能上传视频并创建任务。
- 能配置时长范围、数量限制、片头片尾模板。
- 能使用 `qwen` 生成剪辑方案或候选结果。
- 能产出至少一条可下载的素材文件。
- 能看到任务执行状态和失败原因。
- 不要求完成投放发布闭环。

## 8. 范围说明

如果后续需要扩展到“自动投放、效果优化、跨平台适配”，应作为第二阶段或更后续阶段单独立项，不应混入首版交付范围。
