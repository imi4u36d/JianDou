# 版本记录

## v0.2.2-beta - 2026-03-26

后台管理系统 UI 从“创作工作台风格”重构为“常规管理系统风格”，更适合数据管理和配置排障。

### 主要变更

- 重构后台框架页 `AdminShell`，改为浅色标准管理布局。
- 重构 `AdminDashboardView` 为总览卡片 + 异常任务表 + 可筛选日志表。
- 重构 `AdminTasksView` 为标准筛选工具栏 + 批量操作区 + 数据表格。
- 重构 `AdminTaskDetailView` 为基础信息、规划方案和日志诊断双栏结构。
- 重构 `ModelStatusStrip` 为只读配置表，强化模型/能力项可读性。
- 重构 `AdminSystemView`，支持任务 ID / 级别 / 阶段 / 关键词日志筛选。

## v0.2.1-beta - 2026-03-26

围绕任务创建页完成“四角色三轮迭代”，重点修正参数一致性、模式联动和创建前校验。

### 主要变更

- 前端 `TaskPreset` 类型补齐 `editingMode`、`mixcutContentType`、`mixcutStylePreset`。
- 新建页预设卡片新增模式标识，并按当前模式筛选预设。
- 新增旅行分镜混剪回退预设，补齐混剪默认入口。
- 修复短剧模式多素材冲突：
  - 短剧模式下追加多素材会自动切换为混剪模式。
  - 切回短剧模式会自动保留主素材并移除额外素材。
- 新建页新增“提交校验”清单（标题、素材数量、时长区间、产出数量）。
- 提交按钮与校验状态联动，未通过时禁用提交并提示失败项。
- 文档同步：
  - `docs/UI_UX.md` 更新真实字段、模板枚举与流程说明。
  - `docs/API.md` 更新创建任务示例和模式约束。
  - 新增 `docs/TASK_CREATION_ROLE_ITERATIONS.md` 记录三轮迭代结果。

## v0.2.0-beta - 2026-03-25

基于多 agent 协同完成三轮连续产品迭代，补齐任务工作台、任务复用和模板化建单能力。

### V1：任务可见性增强

- 后端任务详情新增规划方案 `plan`、素材摘要 `source`、开始/结束时间、重试次数、已完成输出数。
- 前端任务详情页新增阶段卡片、规划方案区、素材元数据展示。
- 任务卡片和状态组件升级为更高信息密度的工作台样式。

### V2：任务管理与复用

- `GET /api/v1/tasks` 支持 `q`、`status`、`platform` 过滤。
- 新增 `POST /api/v1/tasks/{taskId}/clone`，返回可用于新建页预填的任务草稿。
- 任务列表页支持搜索、状态筛选、平台筛选、按生命周期分组展示。
- 列表页和详情页都支持“复制参数”进入新建流程。

### V3：模板化建单与工作台指标

- 新增 `GET /api/v1/presets`，提供 4 个静态任务预设。
- 新建页新增预设模板卡片、参数摘要、历史任务复制提示。
- 任务列表页新增总任务、进行中、已完成、失败四类概览指标。
- 筛选状态同步到 URL query，方便回放和分享当前视图。

### 后续五个版本方向

- V4：模型就绪层。补充模型配置校验、运行时状态展示、失败原因细分，不做真实 token 自检。
- V5：字幕驱动切片。突出 SRT/VTT 与时间戳台词输入，增强语义切点规划。
- V6：批量运营工作台。支持按素材和模板批量创建、筛选与复用。
- V7：人工审核与微调。把规划方案从只读展示升级为可编辑、可重渲染。
- V8：策略复盘台。按模板、平台、失败阶段和输出结果做轻量策略分析。
